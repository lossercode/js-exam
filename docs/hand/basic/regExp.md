# 正则表达式

### 分类

- 锚点：`^`, `$`
- 字符组：`[a-z]`
- 零宽断言
- 组和引用
- 分支


### 重难点

- `[^a]`: 否定字符集，表示不是a的字符
- `(?=p)`: 正向先行断言，表示p前面的字符
- `(?!p)`: 负向先行断言，表示p前面的字符
- `(?<=p)`: 正向后发断言，表示p后面的字符
- `(?<!p)`: 负向后发断言，表示p后面的字符
- `(?:)`: 非捕获组
- `+`：一个或多个
- `*`：零个或多个


### 例题

```javascript
// 匹配标签里面的文本
"<div>这是一个div标签</div>".match(/(?<=>).+(?=<)/g)

```
